# MiniRacerSim

## 🚀 最新リリース

**v0.3-beta 公開中！**

ダウンロードはこちら：

https://github.com/aotechlab/MiniRacerSim/releases

![demo](docs/demo.gif)

MiniRacerSim は、ミニ四駆（1:32スケールレーサー）の挙動をシミュレーションおよび解析するためのプラットフォームです。

実車を製作する前に、車両性能、ラップタイム、コース上での挙動、各種セッティング変更の効果を可視化・評価できます。

## 🚀 主な機能

### 車両セッティング

* モータープリセット選択
* バッテリープリセット選択
* ギヤ比設定
* タイヤ径設定
* 車両重量設定
* ローラー性能シミュレーション

### シミュレーション

* インタラクティブなコースビューア
* リアルタイム車両位置表示
* オンボードカメラビュー
* ラップタイムオーバーレイ
* 速度 / RPM / モーター温度ゲージ
* クラッシュ位置表示

### 解析機能

* ラップごとの結果表示
* 最低速度 / 最高速度解析
* カスタムコース評価
* セッティング比較支援

## 📦 ダウンロードと実行方法

1. 最新リリースのZIPファイルをダウンロード
2. ZIPファイルを展開（解凍）
3. MiniRacerSim.exe を実行
4. ブラウザが自動的に起動します

起動しない場合は以下へアクセスしてください：

http://127.0.0.1:8000

---

## ⚠️ 注意事項

* ZIPファイル内のEXEを直接実行しないでください
* 必ずZIPを展開してから実行してください
* Windowsのセキュリティ警告が表示される場合があります

以下をクリックしてください：

**詳細情報 → 実行**

* 初回起動には数秒かかる場合があります
